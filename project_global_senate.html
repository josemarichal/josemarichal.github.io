Collaborative Rule Maker

An AI-Powered Tool for Participatory Policy Design

Project Overview

The Collaborative Rule Maker is an interactive tool designed to help communities design, test, and refine rules through case-grounded deliberation. It helps students explore policy boundaries by generating realistic scenarios and suggests revisions based on feedback.

When given a policy problem, the system brainstorms alternative options, generates "allowed/disallowed/ambiguous" cases for stress-testing, and provides structured revisions to improve clarity and fairness.

How It Works

The application uses an Agentic Mixture of Experts approach, leveraging Google Gemini to simulate a global advisory board. It orchestrates a sophisticated research and design workflow:

  1. User Input: You enter a "wicked problem" (e.g., "Combating urban heat islands" or "Regulating gig economy labor").
  2. Global Perspective Dispatch: The system simulates 10 distinct regional experts (from East Asia to the Nordics) to find concrete success stories.
  3. Agentic Refinement: The system applies advanced reasoning techniques—Iterative breakdown, Cross-Pollination (analogies), and Adversarial review—to stress-test the findings.
  4. Synthesis: A final report identifies consensus best practices, innovative outliers, and a "Politics of the Possible" prototype.

Usage Instructions

To use the tool effectively:

  1. Click the "Launch" button above to open the Gemini Gem interface.
  2. Enter your query in the text box. Be specific for better results (e.g., instead of "Education," try "Vocational training programs for at-risk youth").
  3. Analyze the Results. The Gem will provide regional perspectives, a synthesized report, and an adversarial critique.
  4. Reflect on the "Politics of the Possible." Use the output to identify how local constraints might be overcome using global insights.
